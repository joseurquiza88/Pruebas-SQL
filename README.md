# 📊 SQL Queries

Este proyecto **busca explorar y aprender SQL** mediante la ejecución de consultas sobre distintos datasets.

---

## 🎯 Objetivos

Los datos utilizados en este repositorio **no corresponden a un caso real**, sino que provienen de distintas fuentes (cursos, charlas, workshops, etc.).

La idea fue usar estos datos para **practicar queries en SQL** y:
- Entender mejor la lógica de las consultas  
- Familiarizarme con distintos tipos de datos  
- Ir ganando práctica para poder aplicarlo en un trabajo real  

---

## 🔹 Prácticas de SQL

- Limpieza y preparación de datasets antes de trabajar con ellos  
- Uso de consultas como `SELECT`, `WHERE`, `GROUP BY`, `JOIN`, etc.  
- Pruebas de agregación y filtrado  
- Comparación de distintas formas de resolver una misma consulta  

---

## 🛠️ Herramientas utilizadas

- SQL Workbench para hacer y probar las queries  
- Python (Jupyter Notebook) para explorar los datos y validar algunos resultados
